# Database Integration Project

A Full Stack Development project demonstrating Database Integration using Node.js, Express.js, and SQLite.

## Features
- Database Connection
- Create Data
- Read Data
- Update Data
- Delete Data
- REST API Development

## Technologies Used
- Node.js
- Express.js
- SQLite

## API Routes
- GET /
- GET /contacts
- POST /contact
- PUT /contact/:id
- DELETE /contact/:id
